# NetLogin
A Python scripy used to login Tsinghua net.
```
usage: login.py [-h] [--username USERNAME] [--password PASSWORD] action

positional arguments:
  action               Choose login/logout

optional arguments:
  -h, --help           show this help message and exit
  --username USERNAME  Input your username. if no username specify, use last one
  --password PASSWORD  Input your password. if no password specify, use last one
```
